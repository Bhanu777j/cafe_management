# Cafe Management System

This is a simple restaurant ordering web application.

## Technologies Used
- Python
- Flask
- MySQL
- HTML
- CSS

## Features
- Select food items
- Enter quantity
- Bill calculation
- Stores order data in MySQL database

## Run the Project

1. Install requirements
pip install -r requirements.txt

2. Run the application
python app.py

3. Open in browser
http://127.0.0.1:5000
